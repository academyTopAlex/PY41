# инструкция для md

## выделение текста
для выделения курсивом обрами * вот так: *текст*

для выделения жирным обрами 2 * вот так: **текст**


## работа с листами

для ненумерованных списков используй * так:
* пункт1
* пункт2
* пункт3

для нумерованных списков используй цифры так:
1. пункт1
2. пункт2
3. пункт3
